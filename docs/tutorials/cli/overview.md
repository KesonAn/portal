## 概述

goctl 是 go-zero 配套的代码生成工具脚手架，其集成了 Go HTTP 服务，Go gRPC 服务，数据库 model，k8s，Dockerfile 等生成功能。

goctl 读音为 go control [ɡō kənˈtrōl]，它的主要功能是帮助开发者快速生成代码，提高开发效率。
